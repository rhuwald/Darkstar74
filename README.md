# Darkstar74
Darkstar74 - a retrostyle, vectorbased asteroids clond (written in python)
A game, created especially for the Pimoroni Presto with attached QwSTPad.

The tank is steered with **left** & **right**. To fire, press the **A** button. To thrust, press the **X** button.

- [Required Hardware](#required-hardware)
- [Installation](#installation)
- [Pimoroni Presto](#pimoroni-presto)

## Required Hardware
* Pimoroni Presto
* QwSTPad (with id 0)
* Power (battery or AC adapter)

## Installation
1. Download the python file
2. Start darkstar74_presto.py (e.g., with Thonny)
3. To start Darkstar74 automaticaly - rename it to main.py


## Pimoroni Presto
The Pimoroni Presto is a RP2350-powered microcontroller board with a 4" square touchscreen (480x480px).
